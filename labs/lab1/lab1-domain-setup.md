# Lab 1: Active Directory Domain Setup

## Objective
To install and configure Active Directory Domain Services (AD DS) on Windows Server.

## Environment
- Windows Server 2022
- VMware Workstation

## Steps

### 1. Install the AD DS Role
- Open Server Manager
- Add Roles and Features
- Select Active Directory Domain Services

### 2. Promote to Domain Controller
- Click "Promote this server to a domain controller"
- Select "Add a new forest"
- Domain name: lab.local

## Outcome
Successfully configured a Domain Controller and created a new Active Directory forest.
